About Us

Topical Knowledge Base of Doctrines and Rules System will allow user to enter legal doctrines and rules that the user has learned over the years and to enter specific description of doctrine or rule. The system will analyze the keywords that the user enters and provide the relevant result that are found related to the keywords. To ease the use of the system, the system also implements the use of Voice recognition techniques and regular keyword search to accumulate knowledge of any professional in any field. The system uses keyword search technology to enter a professional question and the system will fetch the result related to the keywords from the database and display the result. If the system is unable to answer the question asked by the user, the Data Administrator will update the knowledge base with the question and the detailed answer.
